# @markee/default

Default configuration extension bundle for Markee.

Full Markee documentation at https://markee.dev/

## Introduction

`@markee/default` ships the baseline extension bundle used by Markee:

- default shared assets
- default layouts and layout fragments
- the standard built-in Markee experience other themes and projects build on top of

## Notes

Most users do not need to add this package manually. Markee includes it automatically so projects always start from a consistent base extension set.
