# @markee/pipeline

Client-side Markdown pipeline for Markee built on remark and rehype.

Full Markee documentation at https://markee.dev/

## Notes

This is an internal package used by `@markee/client` and `@markee/runtime`.

This package exposes the client and search pipelines together with reusable Markee remark/rehype helpers, extensions, and Prism integration utilities.
