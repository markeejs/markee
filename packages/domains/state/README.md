# @markee/state

Nanostores-powered state layer for the Markee site generator.

Full Markee documentation at https://markee.dev/

## Notes

This is an internal package used by `@markee/client` and `@markee/runtime`.

This package exports the shared `state` facade together with the router, metadata loaders, search state, navigation tree state, and cache helpers used by Markee sites.
