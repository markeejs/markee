# @markee/elements

Collection of Markee custom elements, UI primitives, and navigation helpers.

Full Markee documentation at https://markee.dev/

## What It Includes

This package exports:

- low-level blocks such as drawers, popovers, selects, collapses, and scroll areas
- site-level elements such as `markee-content`, navigation, search, revision dates, tabs, and version selectors

## Notes

This is an internal package used by `@markee/client`.

The package exports blocks such as drawers, tooltips, tabs, and scroll areas together with site elements for content, navigation, search, revision dates, and version selectors.
