# @markee/types

Shared TypeScript declarations for the Markee ecosystem.

Full Markee documentation at https://markee.dev/

## Introduction

This package provides the global types used across Markee, including:

- `Configuration`
- `Frontmatter`
- `MarkdownFile`
- navigation tree types
- search result types
- Prism theme names

## Typical Usage

You should never need `@markee/types` directly.

`@markee/runtime` already consumes it and exposes useful types for extensions.
