# @markee/runtime

Extension entrypoints and runtime exports for Markee-generated sites.

Full Markee documentation at https://markee.dev/

## Install

```bash
pnpm add @markee/runtime
```

## Notes

This package exports the `extend` helpers, the shared `state` integration, and the base custom-element runtime used by Markee extensions. Most projects consume it transitively through `@markee/cli` rather than installing it directly.
