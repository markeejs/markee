# @markee/client

Browser client application bundled into generated Markee sites.

Full Markee documentation at https://markee.dev/

## Install

```bash
pnpm add @markee/client
```

## Notes

This package contains the client-side app, public assets, and Vite build used by Markee sites. Most projects consume it transitively through `@markee/cli` rather than importing it directly.
