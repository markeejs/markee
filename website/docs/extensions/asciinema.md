# Asciinema

The `@markee/asciinema` extension allows the inclusion of the [Asciinema](https://docs.asciinema.org/) player whenever an image pointing to a
`.cast` file is used.

## Usage

Any image pointing to a `.cast` file will automatically get replaced by an Asciinema player:
